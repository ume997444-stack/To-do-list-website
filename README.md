# 📝 To-Do List Website

A simple and responsive **To-Do List app** built using HTML, CSS, and JavaScript.  
This project helps manage daily tasks with a clean UI and basic functionality.

## ✨ Features
- ➕ Add new tasks  
- ✅ Mark tasks as completed  
- 🗑️ Delete tasks  

## 🧪 Tech Stack
- HTML5  
- CSS3  
- JavaScript
